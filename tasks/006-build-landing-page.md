## Objective

Build a responsive landing page for the AI Company Demo web app.

## Requirements

### Sections
1. **Hero** — Headline, subtitle, and a call-to-action button
2. **Features** — 3 feature cards highlighting key benefits
3. **How It Works** — 3-step workflow explanation (human -> pm -> engineer/qa)
4. **Footer** — Copyright, links

### Technical
- Semantic HTML5 (`<header>`, `<section>`, `<footer>`)
- Clean CSS (no frameworks required, but keep it maintainable)
- Fully responsive: mobile (< 768px), tablet, desktop
- Place output in `public/index.html` and `public/styles.css`

## Acceptance Criteria
- [ ] All 4 sections render correctly in a browser
- [ ] Page is responsive across mobile, tablet, and desktop
- [ ] Semantic HTML elements used throughout
- [ ] No broken links or missing assets
