## Objective

Build a responsive landing page for the AI Company Demo web app.

## Details

Create a single-page landing page with the following sections:

1. **Hero Section** — Eye-catching headline, subtitle describing the product, and a primary call-to-action button.
2. **Features Section** — 3 feature cards highlighting key benefits (e.g., "AI-Powered", "Coordinated Roles", "Task Automation").
3. **How It Works Section** — Simple 3-step visual flow showing how the system works (Human -> PM -> Engineer/QA).
4. **Footer** — Copyright, basic links.

## Technical Requirements

- Use HTML, CSS (vanilla or with a simple framework like TailwindCSS via CDN)
- Must be responsive (mobile, tablet, desktop)
- Clean, modern design with good typography and spacing
- Place files in a `public/` or `src/` directory at project root
- Ensure the page is servable as a static file

## Acceptance Criteria

- [ ] Hero section with headline, subtitle, and CTA button
- [ ] Features section with at least 3 feature cards
- [ ] How It Works section with workflow steps
- [ ] Footer with copyright
- [ ] Page is responsive across screen sizes
- [ ] Clean, valid HTML and CSS
